What It Does

BUILT FOR
YOUR CREW

Everything a group of people needs to communicate — chat, voice, video, and a moderation toolkit that puts you in charge.

💬

Text Chat

Real-time messaging with all the features you'd expect, and none of the corporate overhead.

  • Threaded replies
  • Emoji reactions
  • File attachments & GIFs
  • Link previews
  • Full message search
  • Unread + mention badges
  • DMs & Group DMs (up to 10)
  • Block users in DMs
  • Slowmode per channel
🎙

Voice & Video

Crystal-clear voice with hardware-grade audio compression and live streaming built in.

  • Push-to-talk (global key hook)
  • Opus audio encoding
  • 48 kHz · 16-bit · mono
  • Screen share (MJPEG)
  • Webcam streaming
  • Adaptive quality control
  • Multi-streamer support
  • Whisper STT transcription
🛡

Admin Suite

Web-based dashboard with full moderation controls and real-time server insight.

  • Member / Moderator / Admin roles
  • Word filter: leet-speak + severity levels
  • Anti-spam & duplicate message automod
  • Warning & strike system
  • Temp bans with auto-expiry
  • @everyone / @role mention gating
  • Per-channel permissions
  • Audit log with retention policy
  • Webhook delivery
Deep Dive

EVERY FEATURE
THAT MATTERS

🔐

Session Management

JWT auth with token blacklisting and server-side session tracking. Sign out of all devices at once. Full IP audit trail on every login. Password reset via secure token — no SMTP required.

Security
🗂

Channel Management

Admins can create, rename, and delete channels on the fly. Set per-channel topics, NSFW flags, slowmode timers, and minimum roles for read and write — all enforced server-side.

Admin
🤖

Whisper Transcription

AI-powered speech-to-text auto-posts what was said in voice into the text channel. Great for logging, accessibility, or just catching what the quiet guy mumbled.

AI
🔗

Invite System

Generate invite links with configurable expiry and use limits. Admins can set vanity codes like /invite/fatguys — cleaner URLs that are actually shareable.

Community
👑

Admin Role Controls

Right-click any member to elevate them to Admin or demote them. Three-tier role hierarchy — Member, Moderator, Admin — with demotion guards so you can't lock yourself out.

Moderation
📊

Live Dashboard

The web admin panel shows online users, voice occupancy, active streams, recent messages, full audit trail, channel management, and word filter controls in real time.

Admin
🌐

Webhook Delivery

Configure per-server webhooks to POST events — new messages, member joins, bans — to any external URL. Build your own bots or pipe events into Discord ironically.

Integration
🏷

Channel Permissions

Set minimum roles for reading and writing per channel independently. Lock a channel to Moderators or Admins. All enforced server-side — not just hidden in the UI.

Moderation
📨

Group DMs

Direct messages and group conversations with up to 10 participants. Same rich message format as channels — reactions, replies, attachments, the works.

Messaging
🔇

Word Filter Severity

Three escalating actions per filter word: Log (replace with asterisks), Delete (block the message outright), or Mute (block + auto-mute the sender for 10 minutes). Leet-speak evasion caught server-side.

Moderation
📣

Mention Gating

Set the minimum role required to use @everyone on a per-server basis. Prevent regular members from pinging the whole server — only Mods or Admins can blast everyone.

Zero Data Mining

No analytics, no telemetry, no ad targeting. Your conversations stay between your crew and your hardware. Period. Nothing phones home.

Discord can ban you, change their TOS, or just go down. FatGuysSpeak runs on your machine. It's live when you want it live — nobody can pull the plug on you.

Custom invite codes, per-channel permissions, word filters, webhook integrations, audit logs. You set the rules, you run the moderation. No algorithm decides what's allowed.
